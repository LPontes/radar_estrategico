# Radar Estratégico™
*Fluência Analítica | Lucas Pontes*

**Radar Estratégico™** é um evento de 3 encontros práticos ao vivo desenhado para profissionais que precisam sair do dado bruto e chegar ao insight executivo — usando o **Framework EDA** aplicado a desafios reais de negócios, sem precisar programar.

## 🗓️ Cronograma dos Encontros (19h00)

* **Aula 1 (23/04) — Construir a Fundação:** Engenharia de dados na prática. Como transformar uma base bagunçada em uma base analítica usando Spec Driven Development (SDD) e ETL.
* **Aula 2 (30/04) — Expandir o Repertório:** Métricas de elite e IA para gargalos. Como aplicar análises de Funil e Churn no seu contexto para apresentar o que a liderança precisa ouvir.
* **Aula 3 (07/05) — Ligar o Radar:** Visão de futuro. Como manter a evolução viva e aplicar o método no seu dia a dia.

## 🎯 O Método: Framework EDA

1. **Foco no Negócio:** Defina a pergunta estratégica antes de abrir os dados.
2. **IA como Motor:** Use a IA para limpeza e cruzamento, enquanto você dirige com contexto.
3. **Sinal nos Dados:** Identifique o detalhe que explica o desafio atual.
4. **Entrega de Valor:** Transforme a descoberta em um *Decision Briefing* (linguagem executiva de 1 página).

> *"Sua pergunta central deve ser: O que eu consigo fazer amanhã de manhã que não conseguia fazer ontem?"*
